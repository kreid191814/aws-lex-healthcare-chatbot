# aws-lex-healthcare-chatbot

## Purpose
This project demonstrates a healthcare workflow chatbot concept using Amazon Lex.

The chatbot is designed to support healthcare operational workflows such as:
- eligibility questions
- customer support routing
- healthcare FAQ assistance

## Skills Demonstrated
- Amazon Lex conversational AI concepts
- API integration concepts
- JSON intent structures
- Workflow automation
- Healthcare systems modernization
- AI-enabled operational support

## Example Use Cases
- “Check my claim status”
- “Where do I upload documentation?”
- “How do I verify eligibility?”
- “Route me to support”

## Technologies & Concepts
- Amazon Lex
- JSON
- API workflows
- Healthcare operational workflows
- AI chatbot architecture
